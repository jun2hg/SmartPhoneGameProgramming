# StickMan Survival
2024-1 스마트폰게임프로그래밍 텀프

1.간단한 소개

High Concept
- 스틱맨을 조종하여 몰려드는 여러적들을 물리치고 강해지면서 더 강한적들을 계속해서 상대하는 2.5D 횡스크롤 슈팅게임

개발일정


1주차 :  UI기획 및 리소스 수집


2주차 : 캐릭터 생성과 이동 구현


3주차 : 몬스터 생성 및 이동 구현

9주차 : 사운드 적용 및 마무리작업
